Las hojas de estilo en cascada tubo sus inicios poco después que el lenguaje de etiquetas SGML, alrededor del año 1970. Desde la creación del lenguaje SGML se considero la necesidad de definir un mecanismo que fuese consistente para la aplicación de estilos a los documentos electrónicos. La invención de esta nueva forma de implementar los estilos en las paginas web, genero un crecimiento exponencial en el lenguaje HTML para la creación de documentos electrónicos, la guerra entre los diferentes navegadores y la necesidad de un estándar que definiera el desarrollo de los estilos en las paginas web dificultaba la creación con la misma apariencia en diferentes navegadores. El organismo W3C (World Wide Web Consortium), encargado de los estándares relacionados con la web, propuso la creación de un lenguaje de hojas de estilo especifico para el lenguaje HTML las cuales se presentaron nueve, de estas se tomaron en cuenta dos que fueron:

CHSS (Cascading HTML Style Sheets)



SSP (Stream-based Style Sheet Proposal)

El propósito de la realización del sitio consta de 2 motivos:

En primer lugar, se desarrolla el sitio como practica para el desarrollo de páginas web utilizando lenguaje HTML para el diseño, estructura y distribución del programa, y para el desarrollo de estilos o de diseño estético utilizando para ello CSS y sus aplicaciones.

En segundo lugar, se crea este sitio como propuesta de creación para un sitio web en donde se implementen temáticas que orienten el desarrollo y conceptualización de temas relacionados con la creación de aplicaciones web, como alternativa de aprendizaje.
