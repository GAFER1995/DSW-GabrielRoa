Introducción a las temáticas

¿Qué es CSS?

Para desarrollar el concepto de CSS es importante tener en cuenta que una pagina web es un documento de texto, codificada con lenguaje HTML, con este lenguaje se pueden crear las estructuras de una pagina web y el contenido que se quiere dar a conocer. La combinación de CSS y HTML permite al diseñador diseñar visualmente una estructura web, no solo basándose en sus niveles jerárquicos sino en su estilo estético, esto permite modificar la estructura web en cuanto a la forma, color, posición y entre otras características visuales a las que se le puede aplicar una página. Las siglas CSS (Cascading Style Sheets) significan "hojas de estilo en cascada", su denominación en cascada hace referencia ya que este concepto se aplica de arriba abajo, el cual sigue un patrón denominado herencia. Para la estructuración de código CSS se utiliza el concepto de separación de presentación y contenido, es decir, se pretende que la estructura codificada con HTML tenga contenida sola información y datos, relativos a la información a transmitir, ya en otro documento CSS se alojan todos los aspectos relacionados con los estilos (diseño, colores,formas,etc…).
