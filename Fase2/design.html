Proposito del sitio

El proposito de la creacion del sitio es principalmente, poder implementar estrategias de aprendizaje por medio de la elaboracion de recursos digitales dinamizados, que puedan facilitar el entendimiento de temas complejos con actividades faciles.

Hojas de estilo en cascada CSS

Según Durango, A. (2015), CSS es el formato recomendado para las páginas escritas en formato HTML en base a los estándares de CSS por sus siglas en inglés (Cascading Style Sheets). La implementación de este estándar web nos permite ahorrar tiempo, da consistencia y facilita en gran medida la escritura de la página web.

A. (2015). Diseño Web con CSS: 2ª Edición. IT Campus Academy.



CSS es uno de los lenguajes bases de la Open Web, esta posee una especialización estandarizada por parte del W3C. Anteriormente las especificaciones de CSS eran realizadas de manera sincrónica, lo que permitía el versionado de las recomendaciones, en donde se especifican CSS1, CSS2, CSS3.

CSS | MDN. (2022, 27 febrero). Mdn. https://developer.mozilla.org/es/docs/Web/CSS

CSS es un lenguaje de hijas de estilo diseñado para el aspecto o presentación de los documentos electrónicos definidos con HTML y XHTML, con la implementación de este formato permite la separación de contenidos y su presentación para la creación depaginas web conplejas.

uniwebsidad. (2006, 1 enero). Capítulo 1. Introducción (Introducción a CSS). https://uniwebsidad.com/libros/css/capitulo-1

Fremeworks CSS

Los frameworks se utilizan comunmente en el ambito de la programacion de aplicaciones, se ha ido implementando los frameworks para programar y diseñar aplicaciones web, es por esta razon que existen cantidades de frameworks para CSS. Se considera un framework como un conjunto de herramientas, librerias, convenciones y buenas practicas que permiten encapsular tareas repetitivas en modulos genericos reutilizables. Los framewoks mas completos incluyen utilidades funcionales, que evitan el trabajar en aspectos genericos del diseño web, de esta manera los framewoks mas completos incluyen herramientas que permitan:

Neutralizar los diseños implementados por defecto de los navegadores.

Manejar correctamente el texto, esto permite que todos los contenidos contengan exactamente los mismos parametros, y favorece la adaptabilidad mejorando su accesibilidad y el acceso a cualquier medio y/o dispositivo.
